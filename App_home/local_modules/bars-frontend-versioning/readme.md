# Модуль Bars.Frontend.Versioning

Данный модуль автоматически забирает метаданные модуля (такие как номер версии и описание) из общего файла и записывает их в package.json проекта.  


## Установка модуля

Модуль устанавливается как и любой другой npm-пакет, командой `npm install bars-frontend-versioning` или `yarn add bars-frontend-versioning`. При установке не через репозиторий, вместо имени пакета, введите путь до модуля.

## Использование модуля

Модуль используется через командную строку. Введите команду `npm run bars-frontend-versioning sync "<путь до файла метаданных>" "<Название модуля внутри файла>"` или `yarn bars-frontend-versioning sync "<путь до файла метаданных>" "<Название модуля внутри файла>"`. Допускается вместо слова `sync` можно писать `s`. Параметр `<Название модуля внутри файла>` необязателен, если его не указывать, то скрипт будет искать строчку по имени пакета, которое было указано в package.json.

## Дополнительные параметры

В дополнение к команде выше, вы можете передать дополнительные параметры.

|Команда         |Описание                       |Примечания                   |
|----------------|-------------------------------|-----------------------------|
|`yarn`          | Использовать yarn classic, вместо npm, для выполнения команд | Описание проекта (description) не будет задано|
